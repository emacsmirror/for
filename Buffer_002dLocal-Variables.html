3 Buffer-Local Variables

All buffer-local variables in this package are provided so that the expansion of certain forms can be configured file-locally. This chapter specifies them.

Variable: for-binder ¶

This buffer-local variable has a symbol as its value that is used as the head of certain let-like forms. The function cell (see (elisp)Function Cells) of the symbol stores either a primitive function (see (elisp)Primitive Function Type) or a macro downward compatible with let. It is an error for it to store any other value. The default value is pcase-let.

For example, a semantics similar to in-ref clauses of cl-loop forms can be achieved by setting for-binder to cl-symbol-macrolet: 

for-binder
     ⇒ cl-symbol-macrolet



(let ((list (list 1 2 3)))
  (for-do ((reference (in-list list))
           (:do (setf reference nil))))
  list)
     ⇒ (nil nil nil)
